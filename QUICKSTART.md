# Quick Start Guide - AFH Consultant Curriculum

Get up and running in 5 minutes!

## Step 1: Install Dependencies

```bash
npm install
```

This installs all required packages including Next.js, React, TypeScript, and Tailwind CSS.

## Step 2: Start Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Step 3: Explore the Curriculum

1. **Homepage** - Overview of all modules and tools
2. **Curriculum** - Start with Module 1: Business Foundation
3. **Practice Exam** - Test your knowledge with interactive questions
4. **Tools** - Access checklists, templates, and calculators
5. **Handbook** - Complete reference guide

## Step 4: Customize (Optional)

- Edit content in `/app/curriculum/[module-name]/page.tsx`
- Modify styling in `tailwind.config.ts`
- Update navigation in `/components/Navigation.tsx`

## Common Commands

```bash
# Development
npm run dev          # Start dev server
npm run build        # Build for production
npm start            # Start production server
npm run lint         # Check for errors
```

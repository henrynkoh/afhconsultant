# AFH Success Consultant Curriculum

A comprehensive Next.js web application designed to train consultants, advisors, and concierges who guide prospective owners through the complete process of setting up and operating financially sound Adult Family Homes (AFH) in Washington State.

## Features

- **Complete Curriculum Modules**: 7 comprehensive modules covering business foundation, site selection, pricing, marketing, billing, medication systems, and resident rights
- **Interactive Practice Exam**: 100-question exam system with explanations and WAC citations
- **Compliance Tools**: Checklists, templates, calculators, and audit tools
- **Master Handbook**: Complete reference guide for consultants
- **Professional UI**: Modern, responsive design with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+ and npm (or yarn/pnpm)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
/app
  /curriculum          # Curriculum module pages
  /exam               # Practice exam system
  /tools              # Checklists, templates, calculators
  /handbook           # Master consultant handbook
/components           # Reusable React components
```

## Curriculum Modules

1. **Business Foundation & Profit Maximization** - Understanding margins and owner's draw
2. **Strategic Site Selection** - Wealth mapping and competitor analysis
3. **Pricing Strategies** - Tiered pricing and competitive positioning
4. **Marketing to Private Pay** - Referral networks and closing strategies
5. **Disclosure & Billing** - WAC-compliant disclosures and invoicing
6. **Medication Systems** - WAC 388-76-10430 compliance
7. **Resident Rights & Services** - WAC 388-10510+ requirements

## Key Features

- **WAC Compliance**: All content based on Washington Administrative Code (WAC) and Revised Code of Washington (RCW)
- **Practical Guidance**: "Cookbook" style instructions for real-world application
- **Financial Focus**: Emphasis on private-pay models and profit maximization
- **Regulatory Defense**: Tools to prevent citations and protect licenses
